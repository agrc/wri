# WRI forklift pallet

## Installation

> [!TIP]
> This pallet has no python dependencies that need to be installed.

This pallet uses the following enterprise geodatabases as sources

- UDNR
- SGID
- UDWRRT2

> [!IMPORTANT]
> Three `*.sde` files need to be installed in the garage for this pallet.
>
> 1. SGID.sde
> 1. UDNR.sde
> 1. UDWRRT2.sde
